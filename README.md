# To-Do-List
To-do lists are a list of tasks that an individual needs to complete or accomplish. <br>
Tasks are typically put in order by priority or importance.<br>

This is an virtual to do list making app where you can organise you work according to your priority and when the work is finished you can clear the work from your list.
